Instagram Clone

I am thrilled to present my Instagram clone project, a dynamic web application that replicates the iconic features of the popular social media platform. This project showcases my proficiency in front-end web development, incorporating front-end technologies to create a seamless user experience.

Key Features:

  • Image Upload: Enabled users to upload images, apply filters, and share their moments with friends and followers, mirroring the core photo-sharing aspect of Instagram.
  • Feed and Profiles: Developed a dynamic news feed where users can view and interact with posts from other users. Each user has a personalized profile showcasing their uploaded content and followers.
  • Instagram Features: Implemented features of real Instagram like design around the profile picture for updated status, comments on posts, etc.

Tech Stack:

  • Front-end: HTML, CSS, JavaScript.
